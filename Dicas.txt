Um avez logado na conta github para atualizar os arquivos modificados arqui, digite:
Quais arquivos estão diferentes:
git status

para subir os arquivos modificados:
git add src/components/app-sections.tsx  -->git add "Nome do arquivo" (ex: src/components/app-sections.tsx)

para fazer o commit no github depois de atualizar os arquivos novos/modificados (git add "arquivo"):
git commit -m "Descrição da mudança"

para compilar:
npm run build

para rodar:
npm run dev
